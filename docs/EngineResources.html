Class: EngineResources

EngineResources()

Storage of all elements used in game like textures, materials, game objects camera, lights etc. use get methods to acquire resources by name

Constructor

new EngineResources()

Creates new engine resources or returns existing one, also creates camera, ambient and direct lights, gui meshes and stores them as resources
